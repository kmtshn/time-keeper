# assets ディレクトリ

このディレクトリにキャラクターの 3D モデルを配置します。

## デフォルトモデル

アプリ起動時、以下のファイルが自動で読み込まれます:

```
assets/character.glb
```

ファイルが存在しない場合は、内蔵のプレースホルダーキャラクターが表示されます。

## 差し替え方法

1. **リポジトリにコミットして反映**: お好みの `.glb` ファイルを `character.glb` の名前でこのディレクトリに置き、push してください。GitHub Pages にデプロイすると、自動的にそのモデルが表示されます。
2. **その場で試す**: アプリ画面の「📁 別の .glb をアップロード」ボタン、もしくはキャンバスへのドラッグ&ドロップで、ローカルの GLB を一時的に表示できます（リポジトリには影響しません）。

## 推奨スペック

- 形式: glTF Binary (`.glb`) 推奨。`.gltf` + 外部リソースも可
- ポリゴン数: 〜数万程度（モバイルでもサクサク動く範囲で）
- 高さ: 任意（アプリ側で自動的に正規化されます）
- アニメーション: 含まれていれば自動再生されます

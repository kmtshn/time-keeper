# 封入封緘タイムキーパー

3Dキャラクターと音声で、封入封緘作業のペースを支援するWebアプリです。
HTML / CSS / Vanilla JavaScript + Three.js (CDN) のみで動作し、ビルド不要です。
GitHub Pages 等の静的ホスティングにそのまま配置できます。

## 主な機能

- **3D キャラクター表示**
  - デフォルトでシンプルなプレースホルダー（青い体＋封筒）を表示
  - ローカルの `.glb` / `.gltf` ファイルをドラッグ&ドロップ or ファイル選択で読み込み可能
  - GLTF にアニメーションがあれば自動再生、無ければゆっくり自動回転
- **タイマー機能**
  - 作業時間 / 休憩時間 / セット数を自由に設定（例: 作業25分 → 休憩5分 × 3セット）
  - 円形プログレスバー & 大きな残り時間表示
  - スタート / 一時停止 / リセット
  - 状態（待機 / 作業中 / 休憩中 / 完了）に応じて背景色が切り替わる
- **音声通知（Web Speech API）**
  - 作業開始時:「作業を開始します。がんばりましょう！」
  - 休憩開始時:「作業終了です！休憩時間だよ！」
  - 全完了時:「すべての作業が完了しました。お疲れ様でした！」
- **励ましボイス（トグルでON/OFF）**
  - 作業時間が半分経過した時:「折り返しだよ。あと〇分！がんばろう！」
  - 残り5分時:「あと5分！がんばろう！」

## 使い方

### ローカルで動かす

ES Modules を利用しているため、`file://` ではなく HTTP サーバ経由で開いてください。

```bash
# Python が入っていれば最も手軽
python3 -m http.server 8000
# その後ブラウザで http://localhost:8000/ を開く
```

### GitHub Pages にデプロイ

1. このリポジトリをそのまま push
2. リポジトリの **Settings → Pages** で `main` ブランチ（または `gh-pages`）をソースに設定
3. 表示された URL でアクセス

外部依存は CDN（unpkg）から読まれる Three.js のみです。

## 動作確認済みブラウザ

- 最新の Chrome / Edge / Safari
- Web Speech API は一部ブラウザ・OS で日本語音声が無い場合があります。  
  日本語音声がない環境では、画面下部の **音声** セレクトから別の音声を選択できます。

## カスタマイズの勘所

- 励ましボイスのタイミング・文言: `script.js` 内 `tick()` 関数の励まし発話部分
- 通知文言: `startWorkPhase()` / `startBreakPhase()` / `finishAll()`
- 背景色: `style.css` の `body.state-work` / `body.state-break` 等
- プレースホルダー形状: `script.js` の `createPlaceholder()`
- 3Dの照明・カメラ位置: `script.js` 上部の Three.js セットアップ

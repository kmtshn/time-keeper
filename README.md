# 封入封緘タイムキーパー

3Dキャラクターと音声で、封入封緘作業のペースを支援するWebアプリです。
HTML / CSS / Vanilla JavaScript + Three.js (CDN) のみで動作し、ビルド不要です。
GitHub Pages 等の静的ホスティングにそのまま配置できます。

## 主な機能

### 3D キャラクター表示
- **デフォルトでリポジトリ内の `assets/character.glb` を自動読み込み**
- ファイル選択 or ドラッグ&ドロップで、ローカルの `.glb` / `.gltf` に一時的に差し替え可能
- 「🔄 デフォルトに戻す」ボタンで元の GLB に復帰
- ファイルが存在しない場合は内蔵プレースホルダーにフォールバック
- GLTF にアニメがあれば自動再生、無ければゆっくり自動回転 + 呼吸風アニメ

### タイマー（2モード切替）

#### ⏱ 時間指定モード
- 作業時間（分） / 休憩時間（分） / セット数 を指定（例: 作業25分 → 休憩5分 × 3セット）

#### 🕐 時刻指定モード（**新規**）
- セットごとに「作業の開始-終了時刻」を個別に指定可能
  - 例: `08:30-09:50`, `10:00-10:50`, `11:00-11:50` ...
- 行の追加・削除が可能、休憩時間は次セットの開始時刻との差から自動算出
- 既に終了時刻が経過したセットは自動でスキップ（視覚的にもグレー表示）
- 「例を入れる」ボタンで一般的な業務時間プリセットを投入

### 音声通知（Web Speech API）
- 作業開始時 / 休憩開始時 / 全完了時に発話
- 励ましボイス（ON/OFFトグル）: 折り返し時 & 残り5分時に発話
- **音声モデル選択**: 利用可能な音声から自由に選択（日本語優先で並び替え）
- **話速・音の高さ調整**: スライダーで微調整可能
- **試し再生ボタン**: 設定確認用に1文を即座に発声

### セリフ編集（**新規**）
すべての通知文言を UI から編集可能。以下のプレースホルダが使えます:

| プレースホルダ | 内容 |
|---|---|
| `{set}` | 現在のセット番号 |
| `{total}` | 総セット数 |
| `{remain}` | 残り分（励まし用） |
| `{end}` | フェーズ終了時刻 (HH:MM) |

例: `セット{set}、作業を開始します。終了予定は{end}。集中していこう！`

「セリフを初期値に戻す」ボタンでデフォルトに復帰できます。

### 設定の永続化
モード・時間・スケジュール・音声・セリフなど、すべての設定は `localStorage` に自動保存されます。
次回アクセス時に同じ設定で立ち上がります。

### UI/UX
- モバイルフレンドリーなレイアウト（2カラム⇔1カラム自動切替）
- 状態別グラデーション背景（待機 / 作業=青系 / 休憩=緑・橙系 / 完了=紫桃系）
- 円形プログレスバー、残り10秒以下でパルス警告アニメ
- 初回タップで音声合成のロックを解除（モバイル対応）

│   ├── character.glb   # ← デフォルトの3Dモデル（任意で配置）

## 使い方

### ローカルで動かす

ES Modules を利用しているため、`file://` ではなく HTTP サーバ経由で開いてください。

```bash
python3 -m http.server 8000
# その後ブラウザで http://localhost:8000/ を開く
```

### GitHub Pages にデプロイ

1. お好みの `.glb` を `assets/character.glb` という名前で配置（無くてもOK、その場合プレースホルダー表示）
2. このリポジトリをそのまま push
3. リポジトリの **Settings → Pages** で `main` ブランチをソースに設定
4. 表示された URL でアクセス

外部依存は CDN（unpkg）から読まれる Three.js のみです。

## カスタマイズの勘所

| 変更したい内容 | 編集箇所 |
|---|---|
| デフォルト GLB のパス | `script.js` の `DEFAULT_GLB_URL` |
| デフォルトのセリフ文言 | `script.js` の `DEFAULT_SCRIPTS` |
| デフォルトの時刻スケジュール | `script.js` の `defaultSettings()` 内 `schedule` |
| 状態別の背景色 | `style.css` の `body.state-work` / `body.state-break` 等 |
| プレースホルダー形状 | `script.js` の `createPlaceholder()` |
| 3Dの照明・カメラ位置 | `script.js` 上部の Three.js セットアップ |
